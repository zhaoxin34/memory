# 家常红烧肉制作指南

## 简介

红烧肉是中国传统名菜，色泽红亮，口感软糯，肥而不腻。

## 主要食材

- 五花肉：500克
- 生姜：3片
- 大蒜：3瓣
- 八角：2个
- 冰糖：30克
- 生抽：2汤匙
- 老抽：1汤匙
- 料酒：1汤匙
- 盐：适量

## 制作步骤

### 1. 处理五花肉

将五花肉切成3-4厘米见方的块状。冷水下锅，加入料酒和姜片焯水，去除腥味。

### 2. 炒糖色

锅中放少许油，放入冰糖小火慢炒。炒至冰糖融化变成琥珀色时，迅速放入五花肉翻炒上色。

### 3. 炖煮

加入生姜、大蒜、八角，倒入生抽和老抽。加入开水没过肉块，大火烧开后转小火炖煮60-90分钟。

### 4. 收汁

待肉质软烂后，大火收汁。不断翻炒使肉块均匀上色，直至汤汁浓稠。

## 技巧要点

1. 选择肥瘦相间的五花肉，口感最佳
2. 炒糖色时要小火慢炒，焦了会发苦
3. 炖煮时一定要加开水，避免肉质变柴
4. 最后收汁要不断翻动，防止糊锅

## 营养价值

红烧肉富含蛋白质和脂肪，适量食用可以补充能量。但血脂较高者应适量食用。
