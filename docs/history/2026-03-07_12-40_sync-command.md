# 变更日志

## 2026-03-07

### 功能新增

- **新增 sync 命令**: 替换原有的 ingest 命令，支持文件的自动同步
  - 自动检测文件的增、删、改操作
  - 基于 MD5 对比判断文件是否变化
  - 支持 `memory sync --repository <repo_name>` 格式调用

- **Repository 实体扩展**: 增加 `root_path` 和 `pattern` 字段
  - `root_path`: 仓库对应的本地文件系统根路径（必填）
  - `pattern`: 用于匹配同步文件的 glob 模式
  - 支持 `memory repo create <name> --root-path /path --pattern "*.md"` 创建仓库

- **Document 实体扩展**: 增加 `relative_path` 字段
  - 存储文档相对于仓库根路径的相对路径
  - 支持相对路径与绝对路径的自动转换

- **repo list 命令增强**: 显示 Root Path 列
  - 方便查看每个仓库对应的本地路径

### 测试更新

- 添加 evaluate 命令到 Makefile
- 更新相关单元测试和集成测试

### 使用方式变化

| 旧命令 | 新命令 |
|--------|--------|
| `memory ingest <file> --repository <repo>` | `memory sync --repository <repo>` |
| `memory repo create <name>` | `memory repo create <name> --root-path <path> --pattern "*.md"` |
