# 股票投资基础知识

## 什么是股票

股票是公司发行的所有权凭证，购买股票即成为公司的股东，享有公司利润分配和重大决策参与权。

## 基本术语

### 市盈率（PE）

股票价格与每股收益的比率，用于评估股票是否被高估或低估。

```
市盈率 = 股票价格 ÷ 每股收益
```

### 市净率（PB）

股票价格与每股净资产的比率。

```
市净率 = 股票价格 ÷ 每股净资产
```

### 分红

公司将部分利润分配给股东，分为现金分红和股票分红。

### 止损

当股价下跌到预定价位时自动卖出，以限制损失。

## 主要分析方法

### 基本面分析

分析公司的财务状况、经营业绩、行业前景等因素，评估股票的内在价值。

**关注指标：**
- 营收和利润增长率
- 毛利率和净利率
- 资产负债率
- 现金流状况

### 技术分析

通过研究股票价格和成交量的历史数据，预测未来价格走势。

**常用工具：**
- K线图
- 移动平均线
- MACD指标
- KDJ指标

## 投资策略

### 价值投资

寻找被低估的优质公司，长期持有，等待价值回归。代表人物：沃伦·巴菲特。

### 成长投资

寻找具有高成长性的公司，虽然估值可能较高，但预期未来利润会大幅增长。

### 趋势投资

顺应市场趋势进行交易，在上升趋势中买入，下降趋势中卖出。

## 风险管理

1. 分散投资：不要把所有资金投入单一股票或行业
2. 设置止损：控制最大损失
3. 保持流动性：不要投资无法承担损失的资金
4. 持续学习：不断提高投资知识和技能

## 常见错误

1. 追涨杀跌：盲目跟风，容易被套
2. 过度交易：频繁买卖，增加成本
3. 杠杆投资：使用借来的钱投资，风险放大
4. 情绪化决策：被贪婪和恐惧左右

## 注意事项

投资有风险，入市需谨慎。建议在充分了解相关知识和风险后，再进行投资。
