# 力量训练完全指南

## 什么是力量训练

力量训练是一种通过抗阻运动来增强肌肉力量和耐力的训练方式。

## 常见训练动作

### 胸部训练

#### 卧推

1. 仰卧在平板凳上，双脚踏实地面
2. 握距略宽于肩，手臂握住杠铃
3. 慢慢放下杠铃至胸部上方
4. 用胸肌力量推起杠铃

#### 哑铃飞鸟

1. 仰卧在平板凳上，双手持哑铃
2. 双手向身体两侧展开，如同拥抱姿势
3. 感受到胸肌拉伸后回到起始位置

### 背部训练

#### 硬拉

1. 双脚与肩同宽，杠铃贴近小腿
2. 屈膝俯身，双手握杠
3. 腿部发力站直，同时挺直背部
4. 慢慢放下杠铃回到起始位置

#### 引体向上

1. 双手握距略宽于肩，挂在单杠上
2. 用背阔肌力量向上拉
3. 下巴过杠后慢慢放下

### 腿部训练

#### 深蹲

1. 双脚与肩同宽，脚尖略微外展
2. 屈膝下蹲，臀部向后坐
3. 大腿与地面平行时站起
4. 保持背部挺直，膝盖不超过脚尖

## 训练计划

建议每周进行2-3次力量训练，每次训练选择2-3个肌群，每个动作3-4组，每组8-12次。

## 注意事项

1. 训练前做好热身，预防受伤
2. 选择合适的重量，循序渐进
3. 保持正确姿势，避免代偿
4. 训练后进行拉伸，帮助肌肉恢复
5. 保证充足睡眠和营养补充
